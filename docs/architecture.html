30-second explanation
  • Factory = rikinptl/web-auto does all the work in GitHub Actions.
  • Inventory = Google Sheet tracks every business and whether it has a live site.
  • Parking lot = each finished site goes to its own kem-llc repo so URLs never overwrite each other.
  • One run can scrape up to N new leads and deploy all of them in parallel.
